Encabezados
Los elementos de encabezado implementan seis niveles de encabezado del documento, h1 es el más importante, y h6, el menos importante
Parrafos
El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.
Imagenes
El elemento de imagen HTML img representa una imagen en el documento.
Enlaces
El elemento SVG a crea un hipervínculo a otras páginas web, archivos, ubicaciones dentro de la misma página, direcciones de correo electrónico o cualquier otra URL.
Tablas
El Elemento de Tabla HTML representa datos en dos o mas dimensiones.
